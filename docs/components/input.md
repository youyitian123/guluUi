# 输入框

## 预览

<ClientOnly><input-demos></input-demos></ClientOnly>

## 文档

:::tip
value : 输入框的值

disabled : 禁用

readonly : 只读

error : 错误提示
:::

## 代码

```html
<g-input value="中文"></g-input>
<g-input value="中文" disabled></g-input>
<g-input value="中文" readonly></g-input>
<g-input value="密码" error="密码错误"></g-input>
```
